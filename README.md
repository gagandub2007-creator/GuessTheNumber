# 🎮 Guess The Number Game (Java)
A simple Java console-based “Guess the Number” game where the program generates a random number (0–99) and the user keeps guessing until correct. Provides hints for higher/lower guesses and uses OOP concepts like classes, methods, and constructors.

## 📌 Description

This project is a beginner-friendly implementation of the classic **"Guess the Number"** game. It demonstrates core Java concepts such as Object-Oriented Programming (OOP), user input handling, loops, and conditional logic.

The program randomly selects a number between **0 and 99**, and the player keeps guessing until the correct number is found. After each guess, the program provides helpful hints.

---

## 🚀 Features

- 🎲 Random number generation using `Random` class  
- ⌨️ User input using `Scanner`  
- 🔁 Continuous gameplay using loops  
- 🔍 Hint system (Too high / Too low)  
- 🧠 Uses OOP concepts (Class, Methods, Constructor)  
- 📊 Tracks number of guesses (property included)

---

## 🛠️ Technologies Used

- Java  
- Java Utility Classes (`Random`, `Scanner`)
